# Sentinela v0.1.2 — versão de apresentação do projeto

A versão 0.1.2 é focada em tornar o projeto mais compreensível para quem acessar o repositório no GitHub pela primeira vez.

## Destaques

- README reestruturado como página de apresentação do projeto.
- Documentação complementar adicionada em `docs/`.
- Screenshots das principais telas adicionados em `assets/screenshots/`.
- Changelog criado.
- Roadmap de evolução incluído.
- Textos atualizados para apresentar a versão atual como protótipo funcional integrado ao banco.

## Funcionalidades já presentes na base funcional

- Cadastro de alunos e tags RFID.
- Cadastro de professores.
- Gestão acadêmica com cursos, turmas e vínculos professor-turma.
- Gestão de ambientes monitorados.
- Gestão de leitores RFID.
- Registro RFID integrado ao banco MySQL.
- Histórico real de movimentações.
- Dashboard com dados reais.
- Mapa com ambientes e leitores reais.

## Limitações conhecidas

- Login ainda visual/local.
- Sem autenticação real no backend.
- Sem perfis e permissões por usuário.
- Sem chamada automática definitiva.
- Sem integração completa com hardware físico.
- Sem relatórios exportáveis.

## Próxima versão planejada

A próxima versão sugerida é a `v0.2.0`, com foco em autenticação real, sessões e perfis de acesso.
